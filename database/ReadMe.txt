setdbconfiguration.ps1
	To configure mongodb docker for the first time run setdbconfiguration.ps1. This will configure the mongo db docker container and start it.

stopdb.ps1
	To stop mongodb docker run stopdb.ps1.
	
startdb.ps1
	To stop mongodb docker run startdb.ps1.

cleardbconfiguration.ps1
	To stop and remove the docker run cleardbconfiguration.ps1
